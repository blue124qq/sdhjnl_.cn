Use of colour

The iDEA colour palette is bright, bold and colourful, with clearly defined category colours, and strong hero colours.

Main colours

These colours can be used interchangeably as background colours and key headline and body text.

Navy
CMYK 100, 90, 32, 22
RGB 35, 47, 94
HEX #2b315c

White
CMYK 0, 0, 0, 0
RGB 255, 255, 255
HEX #ffffff

Accent colours

Use sparingly if content needs to be subtly visually distinguished, but not as starkly as a category.

Muted Grey
CMYK 52, 41, 41, 24
RGB 199, 199, 199
HEX #777777
